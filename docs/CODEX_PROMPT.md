Bu repoyu üretim kalitesinde bir Dava Yönetim ve Strateji Platformuna dönüştür.

Öncelikler:
1. Demo JSON store yerine Supabase Postgres repository katmanı ekle.
2. Supabase Auth oturumunu Expo web/iOS/Android üzerinde çalıştır.
3. Belgeleri private Storage bucket'a kullanıcı/dava klasörleriyle yükle.
4. Belge işleme işini kuyruk tabanlı ve idempotent yap.
5. PDF sayfa numarası, paragraf ve metin koordinatlarını kaydet.
6. Her AI bulgusunu kaynak pasajlara bağla; kaynaksız hukuki cümle üretme.
7. Otomatik araştırma tetikleyicilerini Yargı MCP ve resmi kaynak adaptörlerine bağla.
8. Dilekçe editörüne insan onayı, sürüm geçmişi ve kaynak paneli ekle.
9. Testleri, lint, tip kontrolü, migration ve deployment dokümanını tamamla.
10. API ve MCP için OAuth/organizasyon yetkilendirmesi ekle.

Mevcut kullanıcı akışını ve Türkçe arayüzü koru.
